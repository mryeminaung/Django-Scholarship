# Mini Project: Personal Portfolio

## Overview

`my_portfolio` is a Django project that implements a personal portfolio website. It provides site pages, a projects listing, and a contact information, backed by the `portfolio` app.

## Purpose

This project is intended to showcase work, provide information about me, and show contact information to reach me. It serves as a demonstration of a Django-based portfolio website, with a focus on clean design and responsive layout.

## Key features

- Public pages: Home, About, Projects, Contact
- Projects listing page with per-project details
- Responsive layout and static assets under `portfolio/static/portfolio/`
- Template inheritance in `portfolio/templates/` for consistent layout

## Where to find things

- Templates: `portfolio/templates/` (base.html, home.html, about.html, projects.html, contact.html)
- Static assets: `portfolio/static/portfolio/` (css, images, docs)
- App code: `portfolio/` (views, urls)
- Project settings and entrypoint: `manage.py` and `my_portfolio/`
